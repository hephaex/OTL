# 휴가정책 요약

> 본 문서는 인사규정 중 휴가 관련 내용을 요약한 것입니다.  
> 상세 내용은 HR_Policy_2024.pdf를 참조하세요.

## 1. 연차휴가

### 1.1 연차 발생

| 근속기간 | 연차 일수 |
|:---------|:----------|
| 1년 미만 | 월 1일씩 발생 (최대 11일) |
| 1년 이상 | 15일 |
| 3년 이상 | 17일 |
| 5년 이상 | 19일 |
| 7년 이상 | 21일 |
| 10년 이상 | 23일 |
| 15년 이상 | 25일 (최대) |

### 1.2 사용 방법

1. **사전 신청**: 휴가 시작 3일 전까지 휴가신청서 제출
2. **팀장 승인**: 팀장의 사전 승인 필요
3. **시스템 등록**: 인사시스템에 휴가 등록
4. **반차 사용**: 오전(09:00-13:00) 또는 오후(14:00-18:00) 단위 가능

### 1.3 연차 소멸

- 회계연도 말(12/31) 미사용 연차 소멸
- 이월 불가 (법정 제외)
- 미사용 연차수당 지급

---

## 2. 병가

### 2.1 유급 병가

- 연간 **10일**까지 유급
- 3일 이상 사용 시 **진단서 필수**
- 사후 신청 가능 (복귀 즉시)

### 2.2 무급 병가

- 유급 병가 소진 후 적용
- 최대 30일까지 (휴직 전환 검토)

---

## 3. 경조휴가

| 구분 | 대상 | 일수 |
|:-----|:-----|:-----|
| 결혼 | 본인 | 5일 |
| 결혼 | 자녀 | 2일 |
| 결혼 | 형제자매 | 1일 |
| 사망 | 부모, 배우자 | 5일 |
| 사망 | 배우자의 부모 | 3일 |
| 사망 | 형제자매 | 3일 |
| 사망 | 조부모, 외조부모 | 2일 |
| 출산 | 배우자 | 10일 |

---

## 4. 육아휴직

- **대상**: 만 8세 이하 자녀를 둔 직원
- **기간**: 자녀 1인당 최대 **1년**
- **분할 사용**: 최소 30일 단위로 분할 가능
- **신청 시기**: 시작일 30일 전 인사팀 신청

---

## 5. 기타 휴가

### 5.1 공가

- 병역, 선거, 재판 출석 등 법정 사유
- 증빙서류 제출 필수

### 5.2 특별휴가

- 회사 포상에 따른 휴가
- 장기근속 포상 (10년: 5일, 20년: 10일)

---
